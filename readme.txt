Запустите приложение с Gunicorn:

gunicorn -w 1 -b 127.0.0.1:8000 run:app

Здесь:

-w 4 — количество воркеров (процессов обработки запросов). Вы можете увеличить значение в зависимости от мощности сервера.
-b 0.0.0.0:8000 — IP-адрес и порт, на которых приложение будет доступно.
app:app — путь к Flask-приложению. Первое app — это имя файла (например, app.py), второе app — это объект Flask-приложения.

Распознаваемые штрихкоды:
ZBarSymbol.EAN13        # ISBN, товары
ZBarSymbol.EAN8         # Упрощённые товарные коды
ZBarSymbol.UPC_A        # Товары США/Канада
ZBarSymbol.UPC_E        # Упрощённые UPC
ZBarSymbol.CODE128      # Логистика, здравоохранение
ZBarSymbol.ITF          # Промышленность
ZBarSymbol.CODE39       # Автомобильная промышленность
ZBarSymbol.PDF417       # Почтовые отправления
ZBarSymbol.QRCODE       # QR-коды (исключены)
